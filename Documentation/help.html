Elaborazione foglio presenze

Sviluppato da Guido De Rosa <guidoderosa@gmail.com> Personalizzato originariamente per la A.T.P. srl, Tito, PZ, ITALIA

Può estrarre informazioni da alcuni tipi di file pdf generati dal software PresWeb(R) prodotto da "Selesta Ingegneria".

La configurazione del programma (giorni festivi e dipendenti da escludere) avviene modificando i file escludi.conf e festivi.conf dal menu Configura. È possibile che vi venga chiesto con quale programma si desidera aprire i file di configurazione: scegliete Notepad o un altro editor di testo che eventualmente avete installato.

Nella versione predefinita tali files contengono commenti con istruzioni piu' dettagliate sul loro utilizzo. festivi.conf contiene gia' tutti le festivita' comunemente riconosciute in Italia e le festivita' pasquali fino al 2026. E’ possibile aggiungere, rimuovere o modificare le date senza limitazioni. Si consiglia di salvare una copia di backup dei file di configurazione prima di modificarli.

Requisiti software

  • Microsoft Excel (Microsoft Office). Questo programma e' stato testato con le versioni 2000 e 2003 Pro.
  • Sistema operativo Windows. Il programma e' stato testato su Windows 98 e Windows XP Pro.

Requisiti hardware:

  • E' sufficiente soddisfare i requisiti hardware di Microsoft Office.
